## 주제 : 장애인 콜택시 대기시간 예측

### ✔️ 프로젝트 개요
Why?  
현재 장애인 이동권과 관련된 뉴스가 계속해서 나오고 있다. 특히 콜택시, 지하철, 횡단보도를 이용하는데 많은 어려움이 있다고 한다. 2019년 기준으로 장애인들이 가장 자주 이용하는 교통수단은 택시임이 확인하였고, 따라서 콜택시 대기 시간을 주제로 정하였다.
